# 📱 Mobiles Pi Store & PiTrust Escrow
### سوق الهواتف الذكية الأول والآمن على شبكة Pi Network

تطبيق **Mobiles Pi Store** هو منصة تجارة إلكترونية مبتكرة مصممة خصيصاً لمجتمع Pi، مدمجة بنظام **PiTrust** للضمان المالي (Escrow) لضمان عمليات شراء آمنة وموثوقة بنسبة 100%.

## ✨ المميزات الرئيسية
* **نظام الضمان (Escrow):** يتم حجز عملات Pi بأمان حتى يؤكد المشتري استلام المنتج.
* **واجهة مستخدم عصرية:** متوافقة تماماً مع الأجهزة المحمولة وتدعم اللغة العربية (RTL).
* **تكامل Pi SDK 2026:** استخدام أحدث بروتوكولات المصادقة والمدفوعات لضمان السرعة والأمان.
* **أتمتة ذكية:** ربط كامل مع GitHub Actions لتحديثات النظام الفورية.

## 🛠 التكنولوجيا المستخدمة
* **Frontend:** HTML5, CSS3, JavaScript (Pi SDK v2.0).
* **Backend Logic:** Python (Flask) لعمليات توثيق الضمان.
* **Hosting:** GitHub Pages & Carrd.
